# Tabs

## 🎮 About the Project
A **React-based Tabs component** that renders multiple tabs and allows users to **switch between them dynamically**.  
Designed for seamless navigation and interactive UI experience.

## 🚀 Tech Stack
- **HTML** (BEM methodology for structured markup)  
- **CSS** (Bulma for styling)  
- **JavaScript** (Dynamic logic)  
- **React** (Component-based architecture)  
- **ReactDOM** (Efficient rendering)  

## 🛠️ Tools & Dependencies
- **ESLint** – Ensuring code quality  
- **Prettier** – Code formatting for readability  
- **Cypress** – End-to-end testing  
- **Mochawesome** – Test reporting  
- **Babel** – JavaScript compiler for modern syntax  

## 📌 Features
✅ Renders multiple tabs dynamically  
✅ Allows **seamless tab switching**  
✅ Smooth and interactive **user experience**
